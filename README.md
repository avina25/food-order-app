# React Food Order App

A complete food ordering application built with React to practice advanced React concepts and frontend development workflows.

---

## Features
- Browse meals fetched from a backend
- Add meals to cart
- Increase/decrease cart quantities
- Remove items from cart
- Checkout form with validation
- Submit order to backend
- Loading and error handling
- Responsive UI

---

## Technologies 

- React
- Javascript (ES6+)
- Context API
- CSS
- Fetch API/ HTTP Requests
- Vite
